# Visão geral — Super Embaixadores

## O que é o programa

Programa de parceria com influenciadores jurídicos em duas verticais:

- **OAB** — preparação para exame da OAB
- **ECJ** — Carreira Jurídica (concursos jurídicos)

Cada embaixador pode ter modalidade:

- **Assinatura + Cupom** — acesso ao curso + cupom comissionado
- **Remuneração** — valor fixo mensal proporcional às entregas

## Funil operacional

```
Prospecção (CONTATOS)
  → Candidatura (Forms OAB/ECJ → Respostas)
    → Proposta enviada
      → Formalização (Ativo)
        → Operação mensal (CONTROLE + ENTREGAS + FINANCEIRO)
          → Campanhas pontuais
            → Encerramento (Inativo)
```
